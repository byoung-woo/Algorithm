이 저장소는 **알고리즘 수업** 시간에 진행한 파이썬 실습 코드와 개인 학습 내용을 정리한 공간입니다.
기초적인 제어문부터 시작하여 자료구조(선형 리스트), 재귀 호출 등 알고리즘의 핵심 개념을 코드로 구현하였습니다.

## 📂 학습 내용 및 파일 구성 (Contents)

### 1. Python 기초 및 제어문 (Basics & Control Flow)
파이썬의 기본 문법과 조건문, 반복문을 활용한 실습입니다.
- `1-1.py`: 기본 출력 및 반복문 기초
- `1-2.py`: BMI 계산기 (조건문 `if-elif-else` 활용)
- `1-3.py`: 학점 계산 프로그램
- `1-4.py`: 별 찍기 및 패턴 출력 (중첩 반복문)
- `1-5.py`: 구구단 출력 프로그램

### 2. 자료구조 기초와 함수 (Data Structures & Functions)
리스트, 딕셔너리 등 기본 자료구조의 활용법과 함수의 범위(Scope)를 학습했습니다.
- `2-2 self study.py`: 2차원 리스트 생성 및 조작
- `2-3.py`: 전역 변수(Global)와 지역 변수의 이해
- `2-4.py` / `2-4 계산기.py`: 사칙연산 함수 및 리스트를 활용한 다중 값 반환
- `2-6.py` / `2-6.1.py`: 리스트 함수(`append`, `pop`, `sort`) 및 리스트 컴프리헨션
- `2-7.py`: 딕셔너리(Dictionary) 생성 및 활용

### 3. 선형 리스트와 응용 알고리즘 (Linear Lists & Recursion)
데이터를 순차적으로 저장하는 선형 리스트의 원리와 다항식, 재귀 알고리즘을 구현했습니다.
- **선형 리스트 구현**:
  - `3-1.py` ~ `3-3.py`: 배열 내 데이터 삽입/삭제 시 이동 알고리즘 구현
  - `3-4.py`: 메뉴 선택형 선형 리스트 관리 프로그램 (추가, 삽입, 삭제, 종료)
  - `3-8 ex)1.py`: 정렬된 데이터에 새로운 값 삽입하기 (순위 관리)
- **다항식(Polynomial) 계산**:
  - `3-5.py` / `3-6.py`: 리스트를 이용한 다항식 표현 및 계산 함수(`calcPoly`)
- **재귀(Recursion)**:
  - `3-7 hanoi.py`: 하노이의 탑(Tower of Hanoi) 알고리즘 구현

## 🚀 주요 알고리즘 구현 포인트
1. **데이터 삽입/삭제 로직**:
   - 배열 중간에 데이터를 삽입하거나 삭제할 때, 뒤쪽 데이터를 한 칸씩 밀거나 당기는 로직을 직접 구현하여 메모리 이동 원리를 이해했습니다.
2. **다항식 계산**:
   - `x^n`의 계수를 리스트 인덱스와 매핑하여 다항식 값을 효율적으로 계산했습니다.
3. **재귀 호출**:
   - 하노이의 탑 문제를 통해 복잡한 문제를 작은 단위로 쪼개어 해결하는 재귀적 사고방식을 연습했습니다.
